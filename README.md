## 🚀 Project Overview
This project demonstrates the core practical applications of front-end web development technologies and standard software engineering version control workflows. The website consists of four main functional pages:
1. **🏠 Home:** A premium dark-themed landing page featuring a centralized layout and a brief introduction to blogging.
2. **👤 About Me:** A detailed section highlighting my academic background and skill sets.
3. **📝 Articles (Blog):** Contains 6 short articles about my learning experiences.
4. **✉️ Contact:** A page providing secure channels to connect with me for professional inquiries.

## 🛠️ Technologies & Features Used
1. **HTML:** Utilized semantic tags (`<header>`, `<nav>`, `<article>`, `<footer>`) to construct an accessible web structure.
2. **CSS:** Built a responsive, premium dark UI featuring linear gradients, sticky navigation, custom image scaling, and a flexbox-driven blog sidebar layout.
3. **Git & GitHub:** Implemented industry-standard source code management, tracking incremental commits, and repository organization.

## How to Run the Project
1. 'Clone' or download this repository to your computer.
2. Open the index.html file using any web browser (Google Chrome, Microsoft Edge, etc).
