Profil Saya

Nama saya ialah Yasmin binti Md Shaharuddin.

Saya merupakan seorang pelajar di Fakulti Informatik dan Komputeran (FIK), Universiti Sultan Zainal Abidin (UniSZA). Laman web portfolio ini dibina sebagai bukti pemahaman saya dalam proses pembangunan perisian terutamanya penggunaan kawalan versi seperti Git dan GitHub.

Saya merupakan pelajar UniSZA sejak tahun 2021 apabila saya direzekikan untuk belajar dalam program Diploma Pengajian Islam dengan Teknologi Maklumat dibawah Fakulti Pengajian Kontemporari Islam dan menyambung pengajian juga di UniSZA dalam program Ijazah Sarjana Muda Teknologi Maklumat (Informatik Media).

Antara minat saya termasuklah meneroka teknologi web baharu, menyelesaikan masalah kod, dan membangunkan aplikasi ringkas.
